== Словари: пары ключ-значение ==

Словарь хранит данные по ключам, а не по позициям:

    student = {"name": "Анна", "age": 21, "course": "Python"}
    student["name"]          # "Анна"
    student["city"] = "Казань"   # добавить новую пару
    student["age"] = 22          # изменить значение

Обращение к отсутствующему ключу вызывает KeyError. Безопасный способ - метод get: student.get("city", "не указан") вернёт значение по ключу или значение по умолчанию, если ключа нет.

Ключами могут быть строки, числа и кортежи; списки ключами быть не могут.

ВОПРОС:
Как безопасно достать значение из словаря, если ключа может не быть?

ОТВЕТ:
Используй метод словаря .get(key, default) — он вернёт значение по ключу, а если ключа нет, вернёт default (по умолчанию None) вместо ошибки KeyError:

```python
user = {"name": "Анна"}
age = user.get("age", "не указан")
```

== Частые проблемы ==

«python не распознан как команда» (Windows): Python установлен без галочки Add to PATH. Решение — переустановить Python с этой галочкой либо использовать команду py вместо python.

VS Code не видит Python: нажмите Ctrl+Shift+P, введите «Python: Select Interpreter» и выберите установленную версию 3.12.

Кириллица отображается неправильно при работе с файлами: всегда указывайте encoding="utf-8" в функции open (подробнее — лекция 5.1).

== Критерии оценивания ==

Каждое задание оценивается из 10 баллов:
- 6 баллов — корректность: программа решает задачу на всех проверочных примерах;
- 2 балла — оформление: имена, отступы, докстринги;
- 2 балла — читаемость и структура: отсутствие дублирования, разумное разбиение на функции.

Оценки выставляет проверяющий преподаватель в LMS в течение 5 рабочих дней после дедлайна. Куратор курса оценки не выставляет и не изменяет; вопросы о баллах решаются через апелляцию.

== Апелляция ==

Если вы не согласны с оценкой, в течение 3 дней после её публикации напишите преподавателю через LMS кнопкой «Оспорить оценку» с пояснением, с какими пунктами проверки вы не согласны. Ответ по апелляции приходит в течение 5 рабочих дней.

ВОПРОС:
