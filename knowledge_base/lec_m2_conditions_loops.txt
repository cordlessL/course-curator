ЛЕКЦИЯ 2.1. УСЛОВИЯ И ЦИКЛЫ

== Условный оператор if ==

Условия позволяют программе выбирать, какой код выполнять. Базовая форма:

    temperature = 30
    if temperature > 25:
        print("Жарко")
    elif temperature > 15:
        print("Тепло")
    else:
        print("Прохладно")

Ключевой момент для новичков — отступы. В Python блок кода выделяется не фигурными скобками, а отступом в 4 пробела. Если отступ пропущен или неодинаков, вы получите IndentationError. Настройте редактор так, чтобы клавиша Tab вставляла 4 пробела (см. методичку по настройке окружения, модуль 1).

Веток elif может быть сколько угодно, ветка else необязательна. Python проверяет условия сверху вниз и выполняет только первую подошедшую ветку.

== Логические операторы ==

Условия комбинируются операторами and, or, not:

    if age >= 18 and has_ticket:
        print("Проходите")

Частая ошибка — писать условие «по-человечески»: if x == 1 or 2. Это всегда истинно, потому что число 2 само по себе «истинно». Правильно: if x == 1 or x == 2, либо if x in (1, 2).

== Цикл while ==

Цикл while повторяет блок, пока условие истинно:

    count = 3
    while count > 0:
        print(count)
        count -= 1
    print("Старт!")

Главная опасность — бесконечный цикл: если забыть уменьшать count, программа зависнет. Прервать выполнение в терминале можно сочетанием Ctrl+C.

== Цикл for и функция range ==

Цикл for перебирает элементы последовательности:

    for letter in "python":
        print(letter)

Для повторения действия N раз используется range():

    for i in range(5):      # i пробегает 0, 1, 2, 3, 4
        print(i)

range(start, stop, step) не включает правую границу: range(1, 6) даст числа от 1 до 5. Это одна из самых частых причин ошибок «на единицу» (off-by-one).

== break и continue ==

Оператор break досрочно завершает цикл, continue пропускает текущую итерацию и переходит к следующей:

    for n in range(10):
        if n == 7:
            break        # цикл остановится на семёрке
        if n % 2 == 0:
            continue     # чётные пропускаем
        print(n)         # напечатает 1, 3, 5

== Итоги ==

Блоки кода выделяются отступами в 4 пробела; if/elif/else выполняет первую истинную ветку; while работает по условию, for — по последовательности; range не включает правую границу; break прерывает цикл, continue пропускает итерацию.
