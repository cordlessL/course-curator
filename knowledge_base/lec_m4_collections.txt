ЛЕКЦИЯ 4.1. СПИСКИ И СЛОВАРИ

== Списки: упорядоченные коллекции ==

Список хранит набор значений в определённом порядке:

    fruits = ["яблоко", "банан", "вишня"]

Элементы доступны по индексу, нумерация начинается с нуля:

    fruits[0]    # "яблоко"
    fruits[-1]   # "вишня" — отрицательный индекс считает с конца

Обращение к несуществующему индексу вызывает IndexError. Длина списка — len(fruits).

== Срезы ==

Срез возвращает часть списка: fruits[start:stop:step]. Правая граница, как и в range, не включается:

    numbers = [0, 1, 2, 3, 4, 5]
    numbers[1:4]    # [1, 2, 3]
    numbers[:3]     # [0, 1, 2]
    numbers[::2]    # [0, 2, 4]
    numbers[::-1]   # [5, 4, 3, 2, 1, 0] — перевёрнутый список

== Основные методы списков ==

    fruits.append("груша")     # добавить в конец
    fruits.insert(0, "киви")   # вставить по индексу
    fruits.remove("банан")     # удалить первое вхождение значения
    fruits.pop()               # удалить и вернуть последний элемент
    fruits.sort()              # отсортировать на месте
    len(fruits)                # количество элементов
    "яблоко" in fruits         # проверка наличия, вернёт True/False

Важно: методы append, sort, remove изменяют список на месте и возвращают None. Запись fruits = fruits.append("x") — типичная ошибка, после неё fruits станет None.

== Словари: пары ключ–значение ==

Словарь хранит данные по ключам, а не по позициям:

    student = {"name": "Анна", "age": 21, "course": "Python"}
    student["name"]          # "Анна"
    student["city"] = "Казань"   # добавить новую пару
    student["age"] = 22          # изменить значение

Обращение к отсутствующему ключу вызывает KeyError. Безопасный способ — метод get: student.get("city", "не указан") вернёт значение по ключу или значение по умолчанию, если ключа нет.

Ключами могут быть строки, числа и кортежи; списки ключами быть не могут.

== Перебор коллекций ==

    for fruit in fruits:
        print(fruit)

    for key, value in student.items():
        print(key, "->", value)

Для словаря также доступны student.keys() и student.values(). Если при переборе списка нужен и индекс, и значение, используйте enumerate:

    for i, fruit in enumerate(fruits):
        print(i, fruit)

== Когда что использовать ==

Список — когда важен порядок и элементы однотипны (оценки студента, строки файла). Словарь — когда данные описывают объект с именованными свойствами (карточка студента, настройки). В проекте модуля 4 «Каталог курсов» вы будете комбинировать оба: список словарей.

== Итоги ==

Индексация с нуля, отрицательные индексы — с конца; срезы не включают правую границу; методы списков изменяют его на месте и возвращают None; словари работают по ключам, безопасное чтение — через get; список словарей — базовая структура для табличных данных.
