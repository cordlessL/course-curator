ЛЕКЦИЯ 1.2. ПЕРЕМЕННЫЕ И ТИПЫ ДАННЫХ

== Что такое переменная ==

Переменная — это именованная «коробка», в которую программа кладёт значение, чтобы использовать его позже. В Python переменная создаётся в момент присваивания, объявлять её заранее не нужно:

    age = 25
    name = "Анна"

Знак «=» здесь означает не равенство, а присваивание: «положи значение справа в переменную слева». Имя переменной может состоять из букв, цифр и подчёркиваний, но не может начинаться с цифры. По соглашению PEP 8 имена пишутся в стиле snake_case: user_name, total_price.

== Основные типы данных ==

В Python четыре базовых типа, с которыми вы будете работать в первых модулях:

1. int — целые числа: 42, -7, 0. Python поддерживает сколь угодно большие целые числа без переполнения.
2. float — числа с плавающей точкой: 3.14, -0.5. Помните, что дробная часть отделяется точкой, а не запятой.
3. str — строки: "привет", 'Python'. Кавычки можно использовать одинарные или двойные, главное — одинаковые с обеих сторон.
4. bool — логический тип: True или False. Обратите внимание на заглавную букву.

Узнать тип значения можно функцией type():

    print(type(42))      # <class 'int'>
    print(type("42"))    # <class 'str'>

== Динамическая типизация ==

Python — язык с динамической типизацией: тип привязан к значению, а не к переменной. Одна и та же переменная в разные моменты может хранить значения разных типов:

    x = 10       # сейчас x — int
    x = "десять" # теперь x — str

Это удобно для новичков, но требует внимательности: если функция ожидает число, а получает строку, возникнет ошибка TypeError.

== Преобразование типов ==

Часто данные приходят «не в том» типе. Например, функция input() всегда возвращает строку, даже если пользователь ввёл число. Для преобразования используются функции int(), float(), str(), bool():

    raw = input("Введите возраст: ")  # raw — строка, например "25"
    age = int(raw)                    # теперь age — число 25
    print(age + 1)                    # 26

Если строку нельзя превратить в число (например, int("abc")), Python выбросит ошибку ValueError. Обработку таких ошибок мы разберём в модуле 5.

== Типичные ошибки новичков ==

1. Сложение строки и числа: "Возраст: " + 25 вызовет TypeError. Правильно: "Возраст: " + str(25) или f-строка: f"Возраст: {age}".
2. Сравнение через одно «=»: условие if x = 5 — синтаксическая ошибка. Для сравнения используется двойное равно: if x == 5.
3. Русские буквы и пробелы в именах переменных: технически кириллица допустима, но по стандарту курса имена пишем только латиницей.

== Итоги ==

Переменная создаётся присваиванием; базовые типы — int, float, str, bool; тип проверяется через type(); преобразование — через int(), float(), str(); input() всегда возвращает строку. В следующей лекции разберём операции с числами и строками подробнее.
